Angular - BlockUI Examples

The module blocks the user interface automatically on http request done by Angular but you can still perform blocking manually. Below are some examples on how to do this.

Start and stop block

Start the block manually and stop it after 2 seconds.

Demo 

blockUI.start(); 
$timeout(function() { 
  blockUI.stop(); 
}, 2000);

Block with custom message

Start the block and display a custom message.

Note: the default message can be specified in the configuration.

Demo 

blockUI.start("My custom message");

$timeout(function() {
  blockUI.stop();
}, 2000);

Update messages

Start the block and modify the displayed message.

Demo 

blockUI.start();

$timeout(function() {
  blockUI.message('Still loading ...');
}, 1000);

$timeout(function() {
  blockUI.message('Almost there ...');
}, 2000);

$timeout(function() {
  blockUI.message('Cleaning up ...');
}, 3000);

$timeout(function() {
  blockUI.stop();
}, 4000);

Reset on exception

Example that hides the block whenever an uncaught exception has occured.

Demo 

blockUI.start();

$timeout(function() {
  throw new Error('Oh dear!');
  blockUI.stop(); // Stop will never be called
}, 1000);

Block on HTTP requests

Example that blocks whenever a http request is fired.

Note: The block isn't shown if the server response time is less than the delay set in the configuration.

Demo 

$http.get('/api/quote/').then(function(data) {
  console.log(data);
});
